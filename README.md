# Website Project

This is a website created from WebScrapBook data.

## Structure

- `index.html` - Main homepage
- `css/` - Stylesheets
- `js/` - JavaScript files
- `assets/` - Images, fonts, and other media files
- `pages/` - Additional HTML pages

## Getting Started

1. Replace the content in `index.html` with your WebScrapBook files
2. Add your stylesheets to the `css/` folder
3. Add your scripts to the `js/` folder
4. Store media files in the `assets/` folder

## GitHub Pages

This repository is configured to use GitHub Pages. The site is published at:
`https://birus-des.github.io/und/`

## Next Steps

1. Upload your WebScrapBook website files
2. Go to repository settings → Pages
3. Set the source to "main" branch
4. Your site will be live shortly!
